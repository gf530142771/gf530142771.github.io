# Creator-UI绘制

## 说明
主要通过cocoscreator来拼接界面，导出后直接使用代码生成即可
```
creator.CreatorReader:createWithFilename('.ccreator文件路径')
```

## 导出方法
项目->UIEditor->buildNow

## 其他
1. cocosCreator版本 2.4.5
2. 资源尽量使用图集  
   图集的打法:   
   (1)下载TexturePacker  
   (2)使用工具bingo_newland_app/tools/autopacker.sh来打图集即可 **会有可选路径 记住把资源放进对应的路径**
